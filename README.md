# LASA About-Me Creative Project Website

This is a static website, so it can be deployed to **Vercel**, **GitHub Pages**, or opened locally by double-clicking `index.html`.

## What is included

- Multiple pages/sections: Home, My Story, STEM, Service, Gallery, and Future
- Editable text areas directly on the page
- Profile photo upload
- Favorite quote box
- Image/video gallery upload
- Presentation Mode button that hides editing controls
- Local browser saving with `localStorage`

## How to edit it

1. Open `index.html` in your browser.
2. Click text on the page and type your own wording.
3. Upload a profile photo and gallery media.
4. Click **Save edits**.
5. Click **Presentation Mode** before showing it.

Important: uploads are saved only inside the browser where you added them. For a permanent deployed version, put files in the `assets` folder and update the HTML or use an online media URL.

## Deploy to Vercel

1. Create a GitHub repository.
2. Upload all files from this folder.
3. Go to Vercel and import the GitHub repository.
4. Use the default static site settings.
5. Deploy.

## Deploy to GitHub Pages

1. Create a GitHub repository.
2. Upload all files from this folder. Make sure `index.html` is in the main/root folder.
3. Go to the repository **Settings**.
4. Open **Pages**.
5. Choose the branch, usually `main`, and root folder `/`.
6. Save and wait for the site link.
